L'objectif[modifier]

L'api souhaitée est la suivante :

  • GET /client/: obtenir tous les clients
  • GET /client/[id]: obtenir un client par son id
  • POST /client/: ajouter un client
  • PUT /client/update/[id]: mettre à jour un client par son id
  • DELETE /client/[id]: supprimer un client par son id


Installer django REST framework[modifier]

pip3 install djangorestframework

ou 

apt-get -y install python3-djangorestframework

Créer les sérialiseurs[modifier]

Nous avons besoin d'une classe Serializer pour les instances client afin de gérer la sérialisation vers JSON et la désérialisation à partir de JSON.

- Ce CustomerSerializer héritera de la superclasse rest_framework.serializers.ModelSerializer. - La classe ModelSerializer remplit automatiquement un ensemble de champs et de validateurs par défaut, il suffit de spécifier la classe de modèle.


Créer le fichier serializers.py 

cd ~/workspace/comptaRestApi
vi client/serializers.py

from rest_framework import serializers 
from client.models import Client
 
 
class ClientSerializer(serializers.ModelSerializer):
 
    class Meta:
        model = Client
        fields = ('id',
                  'mnemo',
                  'raison',
                  'adresse1',
                  'adresse2',
                  'codeinsee')
  • model : Spécifie le model relatif au sérialiseur
  • fields: spécifie un tuple de champs que nous souhaitons inclure dans la sérialisation.

Créer les Views[modifier]

from django.shortcuts import render

# Create your views here.
    
from django.shortcuts import render 
from django.http import HttpResponse
from django.http.response import JsonResponse
from django.views.decorators.csrf import csrf_exempt
from rest_framework.parsers import JSONParser 
from rest_framework import status
 
from client.models import Client
from client.serializers import ClientSerializer
 
@csrf_exempt
def client_list(request):
    if request.method == 'GET':
        client = Client.objects.all()
        client_serializer = ClientSerializer(client, many=True)
        return JsonResponse(client_serializer.data, safe=False)
        # In order to serialize objects, we must set 'safe=False'
 
    elif request.method == 'POST':
        client_data = JSONParser().parse(request)
        client_serializer = ClientSerializer(data=client_data)
        if client_serializer.is_valid():
            client_serializer.save() 
            return JsonResponse(client_serializer.data, status=status.HTTP_201_CREATED) 
        return JsonResponse(client_serializer.errors, status=status.HTTP_400_BAD_REQUEST)
 
 
@csrf_exempt 
def client_detail(request, pk):
    try: 
        client = Client.objects.get(pk=pk) 
    except Client.DoesNotExist: 
        return HttpResponse(status=status.HTTP_404_NOT_FOUND) 
 
    if request.method == 'GET': 
        client_serializer = ClientSerializer(client) 
        return JsonResponse(client_serializer.data) 
 
    elif request.method == 'PUT': 
        client_data = JSONParser().parse(request) 
        client_serializer = ClientSerializer(client, data=client_data) 
        if client_serializer.is_valid(): 
            client_serializer.save() 
            return JsonResponse(client_serializer.data) 
        return JsonResponse(client_serializer.errors, status=status.HTTP_400_BAD_REQUEST) 
 
    elif request.method == 'DELETE': 
        client.delete() 
        return HttpResponse(status=status.HTTP_204_NO_CONTENT)


Créer les routes[modifier]

Créez le fichier urls.py pour l'application client 

cd ~/workspace/comptaRestApi
vi client/urls.py

from django.conf.urls import url 
from client import views 
 
urlpatterns = [ 
    url(r'^client/$', views.client_list),
    url(r'^client/(?P<pk>[0-9]+)$', views.client_detail),
]
