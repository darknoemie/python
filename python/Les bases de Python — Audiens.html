# Retourne une liste des nombres premiers de 2 a x
def cribleEratosthenes(x):
    liste = [2]

    # construire une liste avec des nombres impairs apres 2
    for i in range(3, x, 2):
        liste.append(i)

    # supprimer les nombres non premiers
    for i in liste:
     for n in liste:
        if n % i == 0 and i != n:
           liste.remove(n)

    return liste

# Appel de la fonction
print(cribleEratosthenes(20))

Travailler avec les listes

  • list.append(x) : Ajoute un élément à la fin de la liste. Equivalent à a[len(a):] = [x].
  • list.extend(iterable) : Etend la liste list en ajoutant tous les items de iterable. Equivalent à a[len(a):] = iterable.
  • list.insert(i, x) : Insère un élément à la position indiquée. Le premier argument est la position de l’élément courant avant lequel l’insertion doit s’effectuer, donc a.insert(0, x) insère l’élément en tête de la liste, et a.insert(len(a), x) est équivalent à a.append(x).
  • list.remove(x) : Supprime de la liste le premier élément dont la valeur est x. Une exception est levée s’il existe aucun élément avec cette valeur.
  • list.pop([i]) : Enlève de la liste l’élément situé à la position indiquée, et le retourne.
  • list.clear() : Supprime tous les éléments de la liste, équivalent à del a[:].
  • list.index(x[, start[, end]]) : Retourne l'index du premier élément dont la valeur est x. Déclenche une erreur si la valeur n'existe pas. Les arguments start et end définissent une étendue et limitent la recherche à cet ensemble.
  • list.count(x) : Retourne le nombre d’éléments ayant la valeur x dans la liste.
  • list.sort(key=None, reverse=False) : Trie les éléments sur place, (les arguments peuvent personaliser le tri, voir sorted() pour leur explication).
  • list.reverse() : Inverse l’ordre des éléments de la liste, sur place.
  • list.copy() : Renvoie une copie superficielle de la liste (. Équivalent à a[:].
copy.copy (x)  Renvoyez une copie superficielle de x.
copy.deepcopy (x) Renvoyez une copie complète de x.

La différence entre la copie superficielle et la copie profonde n'est pertinente que pour les objets composés (objets qui contiennent d'autres objets, comme des listes ou des instances de classe):

  • Une copie superficielle construit un nouvel objet composé, puis (dans la mesure du possible) y insère des références aux objets trouvés dans l'original.
  • Une copie profonde construit un nouvel objet composé, puis insère de manière récursive des copies des objets trouvés dans l'original.

Travailler sur la tranche

  • Un slice représente une partie de la liste
  • La syntaxe d'un slice est [premier element:dernier element (exclus):le pas]
  • Ainsi stack[0:3:1] retourne les élements d'indice ['zero', 'un', 'deux']
  • et stack[0:3:2] retourne ['zero', 'deux']
>>> stack
['zero', 'un', 'deux', 'trois', 'quatre']
>>> stack[0]
'zero'
>>> stack[4]
'quatre'
>>> stack[2:4]
['deux', 'trois']
>>> stack[0:3]
['zero', 'un', 'deux']
>>>

Le chemin de recherche

L'interpréteur python recherche les modules :

  1. dans le répertoire courant
  2. Dans les répertoires spécifiés dans la variable d'environnement PYTHONPATH.
  3. Dans le répertoire spécifié lors de l'installation.

Les fichiers (17.py)

Ouvrir un fichier

La fonction open permet d'ouvrir un fichier et d'obtenir un descripteur. 

f = open('textfile', 'w')

ou en mode texte 

f = open('textfile', 'wt')


Le mode d'ouverture permet de contrôler l'ouverture du fichier

  • "r" - Lecture - Valeur par défaut. Ouvre un fichier en lecture, génère uneerreur si le fichier n'existe pas
  • "a" - Append - Ouvre un fichier en ajoute, crée le fichier s'il n'existe pas
  • "w" - Write - Ouvre un fichier en écriture, crée le fichier s'il n'existe pas
  • "x" - Create - Crée le fichier spécifié et renvoie une erreur si le fichier existe.

import sys

Ce déplacer dans un fichier

La commande seek permet de déplacer le curseur dans un fichier

Syntaxe: 

fileObject.seek(offset[, whence])
  • offset : déplacement
  • whence : à partir d'où ? (0 par défaut).

whence peut prendre les valeurs suivantes

  • 0 : depuis le début du fichier
  • 1 : depuis la position actuelle
  • 2 : depuis la fin du fichier
# revenir au début du fichier
file.seek(0, 0)
# ..identique à
file.seek(0)
# avancer le curseur de 12 caractères
file.seek(12,1)
# Positionner le curseur 3 caractères avant la fin du fichier
file.seek(-3,2)

Portée des variables (18.py)

  • Sans précision, une variable est locale
  • Une variable déclarée avec nonlocal est reliée à une variable de même nom dans une portée supérieure
  • Un variable global est visible de tout le programme.

Un exemple de code python pour illustrer la portée des différentes portées des variables. 

# Portee des variables
def scope_test():
    stuff = "test stuff"
    def do_local():
        # Sans precision, la variable est locale
        stuff = "local stuff"

    def do_nonlocal():
        # nonlocal relie la variable locale a une variable de portee superieure
        nonlocal stuff
        stuff = "nonlocal stuff"

    def do_global():
        # global indique que la variable est globale
        global stuff
        stuff = "global stuff"

    do_local()
    print("Apres do_local(): stuff="+ stuff) 
    do_nonlocal()
    print("Apres do_nonlocal(): stuff="+ stuff) 
    do_global()
    print("Apres do_global(): stuff="+ stuff) 

scope_test()
print ("Dans le scope global:", stuff)

    def __str__(self):
