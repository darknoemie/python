Introduction

  • Langage interprété, ce qui raccourcit le cycle de développement.
  • Propose une structure de développement.
  • Découpage des applications en modules.
  • Possède des types de données évoluées (tableaux flexibles, dictionnaires, )
  • De nombreux modules existent.
  • Les blocs de code sont définis par l'indentation et pas par des accolades.
  • Par besoin de déclarer les variables ou les arguments de fonctions.
  • Extensible : on peut relier un programme python à du code C

L'interpréteur Python

Sous linux, l'interpréteur se trouve dans : 

/usr/local/bin/python3.5

Ajouter le répertoire /usr/local/bin/ à la variable PATH

Sous windows 

set path=%path%;C:\python35

Lancer python en mode interactif

L'interpréteur
  • L'interpréteur lit l'entrée standard
  • Lorsque l'interpréteur est suivi d'un nom de fichier il prend les intructions dans ce fichier.
