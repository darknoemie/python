Introduction[modifier]

Les interfaces Python de traitement de XML sont regroupées dans le paquet xml.

Les sous-modules de traitement XML sont :

  • xml.etree.ElementTree: l'API ElementTree, un processeur simple et léger.
  • xml.dom: la définition de l'API DOM.
  • xml.sax: classes de bases SAX2 base et fonctions utilitaires.

Vulnérabilités XML[modifier]

Les modules de traitement XML ne sont pas sécurisés contre les attaques XML.

Le site officiel de python donne une liste des attaques : (https://docs.python.org/fr/3/library/xml.html#xml-vulnerabilities)

L'api DOM[modifier]

DOM = Document Object Model

  • DOM présente le document XML comme un arbre.
  • DOM autorise le code client à construire une structure de document depuis zéro.
  • Il permet alors d'accéder à la structure à l'aide d'un ensemble d'objet qui fournissent des interfaces connues.
  • Le DOM (Document Object Model) est défini par le W3C en étapes ou "levels" (niveaux) selon leur terminologie.
  • Le couplage de l'API de Python est essentiellement basée sur la recommandation DOM Level 2
  • Une fois que vous avez un objet document DOM, vous pouvez accéder aux parties de votre document XML à travers ses méthodes et propriétés.

L'api SAX[modifier]

Concepts de base[modifier]

SAX = Simple API for Xml

  • L'analyse d'un document XML par l'API SAX utilise un analyseur XML (parser).
  • Le parser lit le flux XML et réagit à des événements (début d'un élément, fin d'un élément, début du document, fin du document, ...)
  • Un programme SAX est donc une suite de fonctions qui sont déclenchées par l'arrivée d'un événement.
  • L'analyseur SAX ne conserve aucune donnée en mémoire et peut donc être utilisé pour traiter de très gros fichiers.

SAX est un parseur événementiel. les événements sont :

  • startDocument() : survient lorsque le programme démarre la lecture du flux XML.
  • endDocument()  : survient lorsque le programme termine la lecture du flux XML.
  • startElement()  : survient au début d'une balise.
  • endElement()  : survient à la fin d'une balise.
  • characters()  : détection de caractères dans le flux XML.


Analyser XML avec les API SAX[modifier]

SAX est une interface standard pour l'analyse XML pilotée par les événements.

  • Pour analyser XML avec SAX, vous devez généralement créer votre propre ContentHandler en sous-classant xml.sax.ContentHandler.
  • Votre ContentHandler gère les balises et les attributs particuliers de vos saveurs XML.
  • Un objet ContentHandler fournit des méthodes pour gérer divers événements d'analyse.
  • Son analyseur propriétaire appelle les méthodes ContentHandler lors de l'analyse du fichier XML.
  • Les méthodes startDocument et endDocument sont appelées au début et à la fin du fichier XML.
  • Les caractères de la méthode (texte) reçoivent les données de caractères du fichier XML via le texte du paramètre.
  • Le ContentHandler est appelé au début et à la fin de chaque élément.
  • Si l'analyseur n'est pas en mode espace de noms, les méthodes startElement (balise, attributs) et endElement (balise) sont appelées; sinon, les méthodes correspondantes startElementNS et endElementNS sont appelées.


Exemple[modifier]

#!/usr/bin/python3

import xml.sax

class PersonneHandler( xml.sax.ContentHandler ):
   def __init__(self):
      self.CurrentTag = ""
      self.charBuffer = []
      self.paddingCounter = 0

   # Call when an element starts
   def startElement(self, tag, attributes):
      self.CurrentTag = tag
      if tag == "personne":
         print ("*****Personne*****")
         self.paddingCounter=3
         nom = attributes["nom"]
         print ("{0}{1}{2}".format("." * self.paddingCounter,"Nom:", nom))
         numero = attributes["numero"]
         print ("{0}{1}{2}".format("." * self.paddingCounter,"Numero:", numero))
      if tag == "contact-info":
         self.paddingCounter=3
         print ("{0}{1}".format("." * self.paddingCounter,"contact-info"))
          
      if tag == "adresse":
         self.paddingCounter=3
         print ("{0}{1}".format("." * self.paddingCounter,"adresse"))
         self.paddingCounter=6
         pays = attributes["pays"]
         print ("{0}{1}{2}".format("." * self.paddingCounter,"Pays:", pays))
         numero = attributes["numero"]
         print ("{0}{1}{2}".format("." * self.paddingCounter,"Numero:", numero))
         ville = attributes["ville"]
         print ("{0}{1}{2}".format("." * self.paddingCounter,"Ville:", ville))
          

   # Call when an elements ends
   def endElement(self, tag):
      if tag == "contact-info":
         self.paddingCounter-=3
         s = ''.join(self.charBuffer)
         self.charBuffer = []
         print ("->",s,"<-")

   # Call when a character is read
   def characters(self, content):
      self.charBuffer.append(content) 

  
if ( __name__ == "__main__"):
   
   # create an XMLReader
   parser = xml.sax.make_parser()
   # turn off namepsaces
   parser.setFeature(xml.sax.handler.feature_namespaces, 0)

   # override the default ContextHandler
   Handler = PersonneHandler()
   parser.setContentHandler( Handler )
   
   parser.parse("carnet.xml")

import libxml2
import sys

doc = libxml2.parseFile("carnet.xml")
ctxt = doc.xpathNewContext()
racine = ctxt.xpathEval("//*")
for i in range(9):
   print ("racine[{0}]={1}".format(i,racine[i].name))

# Toutes les personnes
personnes=ctxt.xpathEval("//personne")
for i in range(3):
   try:
       print ("personnes[{0}]={1}".format(i,personnes[i].name))
   except IndexError as error:
       print ("Erreur l'index {0} n'existe pas.".format(i))
       break 

# Tous les numero
itemctxt = doc.xpathNewContext()
listitems=ctxt.xpathEval("//personne/contact-info/telephonedomicile")
for item in listitems:
#    itemctxt.setContextNode(item)
    numero=item.xpathEval('.//ItemAttributes/numero')
    print("numero={0}".format(item.prop('numero')))


doc.freeDoc()
ctxt.xpathFreeContext()

Le fichier XSL[modifier]

<?xml version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

<xsl:template match="/">
  <html>
  <body>
  <h2>My CD Collection</h2>
  <xsl:apply-templates/>
  </body>
  </html>
</xsl:template>

<xsl:template match="cd">
  <p>
  <xsl:apply-templates select="title"/>
  <xsl:apply-templates select="artist"/>
  </p>
</xsl:template>

<xsl:template match="title">
  Title: <span style="color:#ff0000">
  <xsl:value-of select="."/></span>
  <br />
</xsl:template>

<xsl:template match="artist">
  Artist: <span style="color:#00ff00">
  <xsl:value-of select="."/></span>
  <br />
</xsl:template>

</xsl:stylesheet>

Le programme xslCatalog.py[modifier]

import lxml.etree as ET

dom = ET.parse('catalog.xml')
xslt = ET.parse('template.xsl')
transform = ET.XSLT(xslt)
newdom = transform(dom)
print(ET.tostring(newdom, pretty_print=True))

outfile = open("catalog.html", 'a')
outfile.write(str(ET.tostring(newdom, pretty_print=True)))
